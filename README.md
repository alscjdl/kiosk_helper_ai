# 📱 디지털 약자를 위한 키오스크 결제 보조 시스템
**[span_0](start_span)[span_1](start_span)2026-1 오픈소스프로그래밍 팀 프로젝트**[span_0](end_span)[span_1](end_span)

## 1. 프로젝트 소개
[span_2](start_span)고령층 및 장애인 등 디지털 약자들이 키오스크 이용 시 겪는 어려움을 해결하기 위해, AI 비전 기술과 생성형 AI를 활용하여 실시간 상황 가이드와 쉬운 설명을 제공하는 서비스입니다.[span_2](end_span)

## 2. 주요 기능
* **이미지 인식**: 키오스크 화면 캡처 이미지에서 텍스트 및 UI 요소 인식 (EasyOCR, YOLO)
* **상황 분석**: 인식된 내용을 바탕으로 현재 결제 단계 분석 및 쉬운 설명 문장 생성 (GPT API)
* **시스템 연결**: 사용자 인터페이스 제작, 음성 가이드 출력 및 전체 모듈 통합 (Streamlit, gTTS)

## 3. 팀원 및 역할
* **윤소희**: 이미지 인식 담당 (EasyOCR, OpenCV, YOLO)
* **조민체**: 상황 분석 및 설명 생성 담당 (GPT API, NLP)
* **김사랑**: UI, 음성 출력 및 시스템 연결 담당 (Streamlit, gTTS)
