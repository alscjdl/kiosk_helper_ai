📱 디지털 약자를 위한 키오스크 결제 보조 시스템

1. 프로젝트 소개
고령층 및 장애인 등 디지털 약자들이 키오스크 이용 시 겪는 복잡한 UI와 어려운 용어로 인한 불편을 해결하고자 함. AI 비전 기술로 화면을 분석하고, 생성형 AI를 통해 사용자 눈높이에 맞는 쉬운 안내와 음성 가이드를 제공하는 서비스임.  

2. 주요 기능 및 사용 기술
• 이미지 인식 (Vision): 키오스크 화면 캡처 이미지에서 텍스트(OCR) 및 버튼/메뉴 등 UI 요소 탐지 (YOLO)
• 상황 분석 및 문장 생성 (AI): 인식된 정보를 바탕으로 현재 결제 단계 분석 및 어려운 용어를 쉬운 표현으로 변환 (GPT API)
• 시스템 통합 및 안내 (App): 사용자 친화적 UI 구성 및 gTTS를 통한 실시간 음성 가이드 출력

3. 팀원 및 역할 분담
• 윤소희: 이미지 인식 담당 (EasyOCR, OpenCV, YOLO 활용)
• 조민체: 상황 분석 및 설명 생성 담당 (GPT API, Prompt Engineering 활용)
• 김사랑: UI 제작, 음성 출력 및 시스템 통합 담당 (Streamlit, gTTS 활용)

4. 개발 프로세스 (Github 기반 협업)
1)	사회적 이슈 조사 및 서비스 설계: 디지털 약자의 불편 사항 분석
2)	데이터 가용성 분석: 키오스크 화면 데이터 및 API 연동 테스트
3)	팀원 역량 분석 및 업무 분장: 기술 스택에 따른 역할 최적화
4)	개발 및 버전 관리: Github Issue 및 Pull Request를 통한 체계적 협업  
5)	결과 검증 및 디버깅: 다양한 키오스크 시나리오별 동작성 확인

5. 기대 효과
• 사회적 가치: 디지털 소외 계층의 정보 접근성 향상 및 삶의 질 개선  
• 기술적 가치: 생성형 AI와 비전 기술을 결합한 실용적인 복합 서비스 구현 경험
